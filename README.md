 Sistema de Registro de Quilometragem – km-nextjs

Aplicação desenvolvida em **Next.js** para registrar, organizar e gerar relatórios em PDF da quilometragem rodada por funcionários durante seus ciclos de trabalho.

O que o sistema faz
- Permite registrar fotos do painel dos veículos como evidência da quilometragem.
- Preenchimento de cabeçalho com:
  - Setor
  - Nome do funcionário
  - Filial
  - Data do ciclo
- Upload das imagens correspondentes às leituras de km.
- Geração automática de um **PDF** contendo:
  - Dados do cabeçalho
  - Quilometragens registradas
  - Datas e evidências fotográficas

Tecnologias utilizadas
- **Next.js**
- **React**
- **TailwindCSS**
- **TypeScript** (se estiver usando)
- **Biblioteca de geração de PDF**

 Objetivo do projeto
Facilitar o processo de comprovação de quilometragem rodada, garantindo organização, evidências visuais e relatórios confiáveis para uso interno da empresa.
